# PyLyricBot
Simple python script to enter genius lyrics into chats

## Dependencies:
* BeautifulSoup
* pyautogui
* requests

## README:
* Script runs the defaults below without arguments
* Usage: lyrics.py URL STARTLINE ENDLINE
* Current timings:
* 3secs until first line, 1/2sec line delay
